1. Difference between flex row and column?

flex-direction is the property which allows to display the contents in a row, column, row-reverse and column-reverse.

flex-direction:row; is used to display all the contents in the horizontal direction. By default, felx-direction value is row.

flex-direction:column; is used to display all the contents in the vertical direction.

for example:
flex-direction:row; flex-direction:column;

2. Explain inline, internal and external stylesheet?

Inline: inline stylesheet is used as style as an attribute with in the tag and is given first preference.

Internal: internal stylesheet is used as a style tag with in the head tag in html file and is given second preference.

External: external stylesheet is used as a separate file with .css extention and is linked with link tag in the html file and is given last preference.

3. Justify-content allows you to do what?

Justify-content property specifies how flex items are distributed along the main axis of the parent container.

flex-start is the default value for Justify-content property.

4. Difference between absolute and relative positioning?

Position:relative; takes its starting position from its original place and leaves its place and overlaps on the next containers as per its values.

Position:absolute; takes its starting position from the top of the web page and overlaps on all the contents as per its values.




5. What is grid-template-columns used for?

grid-template-columns property is used for specifying number of columns in a grid container as well as width of the each columns.

Syntax for 5 columns: grid-template-columns: repeat(3,200px) 150px auto ;

6. What is the z-index in CSS?

z-index property is used to specify the index of different div's i.e, which box should come on the top and bottom while ovelapping.

The value for z-index are ...-2,-1,0,1,2,... etc,

7. What is the Difference between padding and margin?

Padding: padding property is used to provide space around the text or any content in the box as per the size given.

Margin: margin property is used to provide margin around the box as per the size given.
